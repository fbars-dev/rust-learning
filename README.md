# Rust Learning Jouney

Learning Rust from zero - target: smart contract auditing on Solana.

**Background**: Architecture + C + Python + Ruby

**Goal**: Blockchain security, remote work

## Projects

### fizzbuzz
FIrst Rust program. Prints FizzBuzz for numbers 1-100.
Concepts: loops, conditionals, modulo operator

### calcolatrice
Interactive CLI calculator with full error handling.
Concepts: std::io, result, loop, pattern matching, functions.

### lista-spesa
Interactive command-line shopping list.
Concepts: Vec, iterators, enumerate, mutable vs immutable borrowing.

### gestione-contatti
Interactive CLI contact manager with search functionality.
Concepts: struct, impl, methods, iter/filter/collect, borrowing.

### edificio
Struct exercise - building with methods and associated functions.
Concepts: struct, impl, methods, associated functions, derived traits.

### CLI wallet
Exercise on wallet management from CLI. Use of deposit and withdrawal with error management
Concepts: enum, struct, impl, functions, Result, interactive menu.
